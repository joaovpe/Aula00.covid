Operadores 

Relacionais 
Logicos 
Ternario

Relacionais 

> maior
< menor
>= maior ou igual
<= menor ou igual
== igual
!= diferente

identidade 

== apenas testa o valor 

exemplo: 5 == "5" - True poque ele testa apenas o valor da string e do number e não o tipo

=== testa o valor e o tipo

exemplo: 5 === "5" - 1 o primeiro é um number e o segundo uma string ou seja a resposta sera "False"



Logicos 

! Negação
&& Conjunção
|| Disjunção

! true --- false
! false --- true

Conjunção (&&) - Só será verdadeiro quando os dois lados forem verdadeiros ou seja duas necessidades que precisam ser atendidas (e)


Disjunção (||) Basta um ser verdadeiro para ser o resultado ser verdadeiro 
(ou)

Ternário (?) (:) Se e senão

var media = 8

var res = Media >=7 ? "aprovado" : reprovado

res = "aprovado"
